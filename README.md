# Student Registration Form

## Project Description
A simple C# Windows Forms application used to collect student details.

## Features
- Upload student photo
- Enter student name
- Select date of birth
- Choose gender
- Select marital status
- Select qualification
- Select experience
- Enter project details
- Display all entered details in a MessageBox

## Technologies Used
- C#
- Windows Forms
- Visual Studio

## How to Run
1. Open the solution in Visual Studio.
2. Build the project.
3. Run the application.
4. Fill in the student details.
5. Click the Submit button.
